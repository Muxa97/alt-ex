Задание: написать программу, которая при вводе контекстно-свободной грамматики будет определять, задает ли данная грамматика непустой язык и является ли этот язык бесконечным.

Краткая теория
Грамматикой G называют четверку объектов {T, N, P, S} , где T - алфавит терминальных символов, N - алфавит нетерминальных символов (при чем T ∩ N = ∅), P - множество правил вывода, а S - начальный символ грамматики (S ∈ N)
Контекстно свободной называют грамматику, в которой правила вывода имеют следующий вид: A → β (A ∈ N, β ∈ (T ∪ N)⁺)

Полезные и бесполезные символы
Символ называется полезным, если он является порождающим и достижимым, в противном случае он не является полезным и может быть удален из грамматики вместе с содержащими его правилами без изменения языка, пораждаемого этой грамматикой.
Порождающим называется нетерминальный символ A, если существует цепочка вывода A → ... → α, где α ∈ T⁺ (является цепочкой терминалов).
Достижимым называется символ A, если существует цепочка вывода S → ... → αAβ, где S - начальный символ, α, β ∈ (T ∪ N)*.

Проблема пустоты языка
Для определения задает ли грамматика G пустой язык нужно удалить сначала все непорождающие символы и все недостижимые символы. Если после этого остались только эпсилон-правила, то грамматика пораждает пустой язык.

Проблема бесконечности языка
Так же как и для определения пустоты языка, для определения бесконечности языка необходимо удалить непорождающие и недостижимые символы. После этого нужно удалить также все эпсилон-правила. Если после данных преобразований грамматики остается хотя бы одна цепочка вывода вида A → ... → αAβ, где α, β ∈ (T ∪ N)*, то есть, существует "рекурсивный" вывод нетерминального символа A, то язык, пораждаемый грамматикой, бесконечен.
